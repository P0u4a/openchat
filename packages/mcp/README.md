# OpenChat MCP Server

### An MCP server that exposes OpenChat conversations as markdown resources.

### IMPORTANT

You need to first install the [OpenChat extension](https://github.com/p0u4a/openchat) in your browser before using this server.

## Usage

```
npx @p0u4a/openchat-mcp
```

## Options

- `--store`: Path to the conversation store.
- `--bridge-port`: Port to listen for MCP bridge requests.
